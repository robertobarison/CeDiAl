Template
=============================================================================

Le pagine sono basate su template che contengono blocchi di codice html e ciascun blocco riceve la formattazione dal
file css.

Ad ogni blocco viene associato un nome che viene utilizzato anche nel div utilizzato nel css per definire il formato
di visualizzazione.

Tutti template includono altri template fino al file base.html che è il template di base ed è contenuto direttamente
nella directory templates.
