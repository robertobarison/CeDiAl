.. _Oggetto Contatto:

:index:`Contatto <Oggetto Contatto>`
=============================================================================
L'Oggetto Contatto identifica una persone che è il contatto di riferimento di un paziente.

L'oggetto contatto contiene le seguenti informazioni:

- Nome
- Cognome
- Via
- Città
- CAP
- Telefono1
- Telefono2
- Telefono3
- email
